### Project Title
#### An Exploration of US City Bikeshare Data

### Description
The project offers users a simple interactive way to explore different statistics of data from 3 US cities' bike sharing programs, through the command line.

### Files used
- washington.csv
- new_york_city.csv
- chicago.csv
